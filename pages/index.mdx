# La Crypta Wallet Documentation

Welcome to La Crypta Wallet! This is a basic docs template. You can use it as a starting point for your own project &#x1f603;

## What is La Crypta Wallet?

An open source wallet that provides NFC cards functionality.
This documentation talks about the Backend components: SDK and the Wallet Provider.

## Components

### HODL.ar (SDK)

- NIP-05
- LUD-16

### Wallet Provider

- Create Users
- Create Cards
- Holds Wallet data
- Remove Card
- Make payments
- Generates LNURLp
- Emits NIP-05 events on wallet transfer
