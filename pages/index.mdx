# La Crypta Wallet Documentation

## What is La Crypta Wallet?

An open source wallet that provides NFC cards functionality.
This documentation talks about the Backend components: SDK and the Wallet Provider.

## Components

### [HODL.ar (SDK)](./hodl)

- Public Registry
- NIP-05
- LUD-16

### [Wallet Provider](./wallet-provider)

- Create Users
- Create Cards
- List Transactions
- Get balance
- Holds Wallet data
- Remove Card
- Make payments
- Generates LNURLp
- Emits transaction events on nostr on wallet transactions (send/receive)
