# Account

Stores User Data

## Schema

- `id` : UUID
- `name` : String

## Relations

- [Balance](./balance) - MANY
- [BalanceSnapshot](./balance-snapshot) - MANY

## Prisma definition

```prisma
model Account {
  id       String    @map("id") @db.Uuid
  name     String    @map("name")
  balances Balance[] @relation

  @@id([id])
  @@index([name])
  @@map("accounts")
}
```
