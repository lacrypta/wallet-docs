# GET /card/pay

This is the second part of a [LUD-03](https://github.com/lnurl/luds/blob/luds/03.md)

Query parameters:

- **`k1: string`:** the `k1` value sent in the scan response.
- **`pr: string`:** the payment request generated by the POS.
