# MovementType

Declares movement types

## Schema

- `id` : ID
- `description` : String

## Relations

- [Movement](./movement) - MANY

## Prisma definition

```prisma
model MovementType {
  id          String     @map("id") @db.Uuid
  description String     @map("description")
  movements   Movement[] @relation

  @@id([id])
  @@map("movement_types")
}
```
