# Token

Stores Token list

## Schema

- `id` : ID
- `name` : String

## Relations

- [Balance](./balance) - MANY
- [BalanceSnapshot](./balanceSnapshot) - MANY

## Prisma definition

```prisma
model Token {
  id   String @id @default(uuid())
  name String

  balances       Balance[]
  balanceSnapshots BalanceSnapshot[]
}
```
