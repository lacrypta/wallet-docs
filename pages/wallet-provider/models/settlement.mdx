# Settlement

Stores a set of BalanceSnapshots than cancel each other out.

## Schema

- `id` : ID
- `metadata` : String

## Relations

- [BalanceSnapshot](./balanceSnapshot) - MANY

## Prisma definition

```prisma
model Settlement {
  id       String @id @default(uuid())
  metadata String

  balanceSnapshots BalanceSnapshot[]
}
```
